{% trans "Analysis pipeline" %}

{% trans "24 stages, run linearly. Compute-heavy stages (marked ★) run on SLURM cluster nodes." %}

{% trans "GenBank download from NCBI (or manual upload), import to database: proteins, sequences, original annotations." %}

4–7 ★
{% trans "Target scoring" %}

{% trans "Protein annotation" %}

{% trans "Sequence indexing and InterProScan (run in SLURM): domain annotation, GO terms, EC numbers from 20 integrated databases." %}

12–19 ★
{% trans "Structural evidence" %}

{% trans "UniProt mapping, experimental PDB retrieval, AlphaFold DB download, ColabFold predictions, FPocket + P2Rank pocket detection, druggability score loading." %}

{# ── 3. 3D Structure ── #}

{% trans "3D structure" %}

{% trans "Stages 13, 15, 16 · fetch_experimental_structures → alphafold_unips → colabfold_predict" %}

{% trans "Each protein is assigned a single preferred structure for analysis, following a strict priority order:" %}

1
{% trans "Experimental structure (PDB)" %}

{% trans "Crystallography, cryo-EM, or NMR entries from the Protein Data Bank. Retrieved by searching PDB for all structures linked to the protein's UniProt ID. Multiple chains and structures can be available and switched in the viewer." %}

2
{% trans "AlphaFold Database model" %}

{% trans "Computational model downloaded from the AlphaFold Protein Structure Database (EBI/DeepMind) using the UniProt ID. Requires a successful UniProt mapping at stage 12." %}

3
{% trans "ColabFold prediction" %}

{% trans "AlphaFold2-based prediction using MMseqs2-accelerated MSA search, generated locally for proteins without other structural coverage. Produced at stage 16, either on CPU (~30–60 min/protein) or GPU via SLURM." %}

{% trans "pLDDT — model confidence" %}

{% trans "The predicted Local Distance Difference Test (pLDDT) is a per-residue confidence score computed by AlphaFold and ColabFold. The platform stores and displays the mean pLDDT across all residues." %}

≥ 90{% trans "Very high confidence" %}
70–89{% trans "Good confidence" %}
50–69{% trans "Low confidence — possibly disordered region" %}
< 50{% trans "Not reliable — avoid structural interpretation" %}
{# ── 4. Binding sites ── #}

{% trans "Binding sites" %}

{% trans "Stage 17 · structures_remote" %}

{% trans "Two complementary tools detect and rank putative binding pockets on each protein structure. Both are run on SLURM at stage 17 and results are stored as residue sets visualizable in the 3D viewer." %}

FPocket v4+

{% trans "Detects surface cavities using alpha spheres derived from a Voronoi tessellation of the protein surface. For each pocket, computes a druggability score based on pocket volume, hydrophobic residue fraction, solvent accessibility, and dipole moment." %}

≥ 0.7{% trans "Highly druggable" %}
0.4–0.69{% trans "Moderately druggable" %}
< 0.4{% trans "Low druggability" %}

{% trans "The druggability score shown per-protein is the maximum score across all detected pockets for that structure." %}

P2Rank v2.x

{% trans "Applies a random forest classifier trained on ~15,000 PDB structures. For each point on the protein surface, it estimates a probability of belonging to a ligand-binding site using local descriptors: solvent accessibility, hydrophobicity, and evolutionary conservation." %}

≥ 0.5{% trans "High probability binding site" %}
0.2–0.49{% trans "Medium probability" %}
< 0.2{% trans "Low probability" %}

{# ── 6. Target profile ── #}

{% trans "Target profile" %}

{% trans "Stages 4–7, 20–21 · fasttarget → psort" %}

{% trans "The target profile consolidates four evidence signals to help prioritize proteins as therapeutic candidates. All are computed computationally — see the interpretation notes below." %}

{% trans "Human off-target" %}

{% trans "DIAMOND BLASTP against the complete human proteome (UniProt/Swiss-Prot + TrEMBL). A hit is reported when e-value ≤ 1×10⁻⁵. The best-alignment identity percentage and e-value are stored." %}

{% trans "Desirable:" %} {% trans "no hit — reduces the risk of cross-toxicity to the human host." %}

{% trans "Gut microbiome off-target" %}

{% trans "DIAMOND BLASTP against a curated reference set of human gut commensal genomes. A hit requires sequence identity > 40% AND query coverage > 70%." %}

{% trans "Desirable:" %} {% trans "no hit — a compound inhibiting a microbiome-conserved protein may disrupt beneficial gut flora." %}

{% trans "Essentiality (DEG)" %}

{% trans "DIAMOND BLASTP against the Database of Essential Genes — genes confirmed essential by transposon mutagenesis, deletion, or fitness competition in model organisms." %}

{% trans "Desirable:" %} {% trans "hit — a protein with DEG homologs is more likely essential to pathogen viability." %}

{% trans "Subcellular localization" %}

{% trans "PSORTb v3 uses an SVM trained on bacterially characterized proteins to classify localization as Cytoplasmic, CytoplasmicMembrane, Periplasmic, OuterMembrane, Extracellular, or Unknown." %}

{% trans "Context:" %} {% trans "outer membrane, periplasmic, and extracellular proteins are more accessible to large molecules. Cytoplasmic targets require compounds to cross both membranes." %}

{# ── 8. Drug-likeness ── #}

{% trans "Drug-likeness" %}

{% trans "Computed from SMILES using RDKit at page load time. Not experimental values." %}

MW{% trans "Molecular weight (Da)" %}

{% trans "Average molecular weight accounting for natural isotopic abundances. Lipinski et al. propose ≤ 500 Da for oral permeability — though many approved antibiotics exceed this (beta-lactams, glycopeptides)." %}

LogP{% trans "Lipophilicity" %}

{% trans "Crippen's atomic contribution method. Measures octanol/water partitioning. Range 0–3 is generally favorable; > 5 indicates potential toxicity and metabolic issues; < −2 suggests poor membrane permeability." %}

TPSA{% trans "Topological polar surface area (Å²)" %}

{% trans "Sum of polar atom surface areas (O, N, and their hydrogens). < 140 Å² satisfies Veber's bioavailability criterion; < 60 Å² improves permeation across bacterial double membranes." %}

Ro5{% trans "Lipinski Rule of Five" %}

{% trans "Checks four conditions: MW ≤ 500 Da, LogP ≤ 5, H-bond donors ≤ 5, H-bond acceptors ≤ 10. Violations count: 0 = compliant, 1 = caution, ≥ 2 = likely oral bioavailability issues. Applies primarily to oral drugs — parenteral antibiotics routinely violate Ro5." %}

PAINS{% trans "Pan-assay interference" %}

{% trans "RDKit's PAINS filter (Baell & Holloway catalog) flags substructures prone to false positives in HTS via promiscuous reactivity, optical interference, or non-specific binding. A PAINS alert does not disqualify a compound, but requires orthogonal validation." %}

{# ── 9. Interpretation notes ── #}

{% trans "Interpretation notes" %}

  • {% trans "All evidence is computational" %} {% trans "— except PDB co-crystal binders (experimental co-crystallization) and direct ChEMBL entries (measured in vitro/in vivo bioactivity). Everything else is predicted, not confirmed." %}
  • {% trans "Absence of a value ≠ absence of the property." %} {% trans "EC = 0 can mean the protein genuinely has no characterized enzymatic activity, or that the pipeline found insufficient homology, or that the tool was not configured for that organism. Context matters." %}
  • {% trans "Druggability scores are structural descriptors," %} {% trans "not predictors of biological activity or toxicity. A high score indicates a well-defined pocket with favorable physicochemical properties — it does not imply that a known drug exists or that the pocket is the active site." %}
  • {% trans "DEG essentiality is transferred:" %} {% trans "a protein may be essential in the DEG reference organism but not in the studied pathogen, and vice versa. Genome-specific essentiality screens are the gold standard." %}
  • {% trans "AlphaFold and ColabFold models have known failure modes" %} {% trans "in intrinsically disordered regions, membrane proteins, and proteins with few homologs. Do not use model structures for pocket analysis in regions with pLDDT < 50." %}
  • {% trans "Physicochemical properties (MW, LogP, TPSA, Lipinski)" %} {% trans "are computed from SMILES and are predicted values, not experimental measurements." %}
