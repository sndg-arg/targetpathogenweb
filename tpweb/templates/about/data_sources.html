< 0.4{% trans "Low druggability" %}

{% trans "Applies a random forest classifier trained on ~15,000 PDB structures. For each surface point, estimates the probability of belonging to a ligand-binding site using solvent accessibility, hydrophobicity, and evolutionary conservation as descriptors." %}

{# ── 6. Target profile ── #}

{% trans "Stages 4–7, 20–21" %}

{% trans "Target profile" %}

{% trans "fasttarget → psort" %}

{% trans "Four evidence signals are combined to support target prioritization. All are computationally derived — see interpretation notes at the bottom of this page." %}

{% trans "Human off-target" %} DIAMOND BLASTP · UniProt

{% trans "Searches the complete human proteome. A hit is reported when e-value ≤ 1×10⁻⁵. Best-alignment identity and e-value are stored." %}

{% trans "Desirable: no hit — reduces risk of cross-toxicity to the human host." %}

{% trans "Gut microbiome" %} DIAMOND BLASTP · curated gut genomes

{% trans "Searches a curated reference set of human gut commensal genomes. A hit requires identity > 40% AND query coverage > 70%." %}

{% trans "Desirable: no hit — a compound targeting a microbiome-conserved protein may disrupt beneficial gut flora." %}

{% trans "Essentiality" %} DIAMOND BLASTP · DEG

{% trans "Searches the Database of Essential Genes — genes confirmed essential by transposon mutagenesis, deletion, or fitness competition in model organisms." %}

{% trans "Desirable: hit — proteins with DEG homologs are more likely essential to pathogen viability." %}

{% trans "Subcellular localization" %} PSORTb v3

{% trans "SVM classifier trained on experimentally characterized bacterial proteins. Predicts: Cytoplasmic, CytoplasmicMembrane, Periplasmic, OuterMembrane, Extracellular, or Unknown." %}

{% trans "Context: outer membrane and extracellular proteins are more accessible; cytoplasmic targets require compounds to cross both membranes." %}

{# ── 8. Drug-likeness ── #}

{% trans "Computed from SMILES · RDKit" %}

{% trans "Drug-likeness" %}

{% trans "Calculated at page load. Predicted values, not experimental measurements." %}

MW {% trans "Molecular weight (Da)" %}

{% trans "Average molecular weight accounting for natural isotopic abundances. Lipinski et al. propose ≤ 500 Da for oral permeability, though many approved antibiotics exceed this (β-lactams, glycopeptides)." %}

LogP {% trans "Lipophilicity" %}

{% trans "Crippen atomic contribution method. Measures octanol/water partitioning. Range 0–3 generally favorable; > 5 suggests toxicity and metabolic issues; < −2 indicates poor membrane permeability." %}

TPSA {% trans "Topological polar surface area (Å²)" %}

{% trans "Sum of polar atom surface areas (O, N, and their hydrogens). < 140 Å² satisfies Veber's bioavailability criterion; < 60 Å² improves permeation across bacterial double membranes." %}

Ro5 {% trans "Lipinski Rule of Five" %}

{% trans "Four conditions: MW ≤ 500 Da, LogP ≤ 5, H-bond donors ≤ 5, H-bond acceptors ≤ 10. Violations are counted (0 = compliant). Applies primarily to oral drugs — parenteral antibiotics routinely violate Ro5." %}

PAINS {% trans "Pan-assay interference" %}

{% trans "RDKit PAINS filter (Baell & Holloway catalog). Flags substructures prone to false positives in HTS via promiscuous reactivity, optical interference, or non-specific binding. An alert does not disqualify a compound, but requires orthogonal validation." %}

{# ── 9. Interpretation ── #}

{% trans "Important" %}

{% trans "Interpretation notes" %}

  • {% trans "All evidence is computational" %} {% trans "— except PDB co-crystal binders (experimental co-crystallization) and direct ChEMBL entries (measured bioactivity). Everything else is predicted, not confirmed." %}
  • {% trans "Absence of a value is not absence of the property." %} {% trans "EC = 0 can mean the protein has no characterized enzymatic activity, or that the pipeline found insufficient homology, or that the tool was not configured for that organism." %}
  • {% trans "Druggability scores are structural descriptors," %} {% trans "not predictors of biological activity or toxicity. A high score indicates a well-defined pocket with favorable physicochemical properties — it does not imply a known drug exists or that the pocket is the active site." %}
  • {% trans "DEG essentiality is transferred:" %} {% trans "a protein may be essential in the DEG reference organism but not in the studied pathogen, and vice versa. Genome-specific essentiality screens are the gold standard." %}
  • {% trans "AlphaFold and ColabFold models have known failure modes" %} {% trans "in intrinsically disordered regions, membrane proteins, and proteins with few homologs. Do not use model structures for pocket analysis in regions with pLDDT < 50." %}
  • {% trans "Physicochemical properties are predicted from SMILES," %} {% trans "not measured experimentally. Lipinski Ro5 is a statistical guide for oral drugs — not an absolute rule." %}
